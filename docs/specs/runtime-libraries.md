# 実行時ライブラリ

初期アプリ実装では、README に記載された白い正四面体と星空背景を描画するため、次のライブラリを導入します。

## React

UI のエントリーポイントとブラウザイベント管理に使用します。

## Three.js

正四面体、星空の点群、ライト、カメラなど、3D 描画の中核に使用します。

## @react-three/fiber

React コンポーネントとして Three.js のシーンを構築し、描画ループを管理するために使用します。

## @react-three/drei

React Three Fiber 向けの補助コンポーネントを利用するために使用します。初期実装ではカメラ設定とアセット事前読み込みに使います。

## Vitest

WebGL 対応確認、ランダム回転軸、星空座標生成など、描画前に検証しやすい関数のテストに使用します。

## Playwright

開発サーバー上で実際のブラウザを起動し、Canvas のスクリーンショットとピクセル状態を確認するために使用します。
