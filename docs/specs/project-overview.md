# プロジェクト概要

`sandbox-webcg` は、ウェブページで CG を描画するための技術である WebGL と WebGPU に関する実験を行うプロジェクトです。

主な目的は、同じ描画課題を WebGL と WebGPU で実装し、API 設計、シェーダー言語、レンダリングパイプライン、リソース管理、ブラウザ互換性の違いを比較できる状態にすることです。
