# プロジェクト概要

`sandbox-webcg` は、ウェブページで CG 表現を試すための実験プロジェクトです。

主な目的は、同じ描画課題を WebGL と WebGPU で実装し、API 設計、シェーダー言語、レンダリングパイプライン、リソース管理、ブラウザ互換性の違いを比較できる状態にすることです。

## 現在の状態

2026-05-26 時点では、最初の実装目標として次の画面を想定しています。

- 画面アクセス時に、黒い星空の中へ白い正四面体を表示する。
- 正四面体はアクセスするたびにランダムな方向へ回転する。
- 背景の星空は白い点を散らした天球として表現し、正四面体に合わせて回転する。
- ドラッグ操作により、正四面体と星空を任意方向に回転できる。

このドキュメント群は、今後 Codex や開発者が実装を追加するときの前提情報として作成されています。

## 使用技術

基本の使用技術は次の通りです。

- Vite
- TypeScript
- React
- ESLint
- Prettier
- Three.js
- Vitest

`@react-three/fiber`、`@react-three/drei`、`gsap` など、描画やアニメーションの実装に有用なライブラリは柔軟に採用できます。

新しい主要ライブラリを導入した場合は、導入した技術名と導入目的を `docs/specs/` に記録します。
