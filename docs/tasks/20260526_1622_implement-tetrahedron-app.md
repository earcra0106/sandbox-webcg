# implement-tetrahedron-app

## 作業タイトル

feature/implement-tetrahedron-app

## 作業の目的と概要

README に記載された初期画面を実装する。

ブラウザでページにアクセスすると、黒い星空の中に白い正四面体が表示され、正四面体と背景の星空がランダムな方向に回転する。ドラッグ操作により、正四面体と星空を任意方向に回転できる。

## 作成・編集するファイルのスコープ

- Vite、React、TypeScript、Three.js を使うアプリ実装ファイル
- ESLint、Prettier、Vitest の設定ファイル
- 導入ライブラリの目的を記録する docs
- 実装作業のタスク記録

## 実装する関数、クラス、モジュールの概要

- WebGL 対応可否を確認する関数
- ランダムな回転軸を生成する関数
- 星空の点群座標を生成する関数
- 正四面体と星空背景を描画する React コンポーネント

関数を作成する場合は対応する spec ファイルを先に作成し、通ることを確認してから実装する。

## 実装した機能にアクセスするための URL パスや UI 操作の説明

- URL: `/`
- 操作: 画面をドラッグして、正四面体と星空背景を任意方向に回転する。
